Javascript

Apa itu Javascript?

Javascript merupakan bahasa skript populer yang dipakai untuk menciptakan halaman web yan dapat berinteraksi dengan pengguna serta dapat merespon event yang terjadi pada halaman. kombinasi HTML, CSS, dan Javascript dapat menghasilkan halaman web yang terstruktur, bergaya ,dan interaksi

Fungsi

Beberapa fungsi Javascript di antaranya yaitu : mendeteksi event=event yang disebabkan oleh pengguna, menvalidasi apa yang diketikkan pengguna melalui form sebelum dikirim ke server, dan masih banyak lagi.
